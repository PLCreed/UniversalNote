# UniversalNote

### Quick Start
```shell
git clone git@github.com:PLCreed/UniversalNote.git
cd UniversalNote
git submodule init
git submodule update
```
### Add Submodule
```shell
git submodule add git@github.com:xxx/my_common.git  
```

### Update Submodule
```shell
git submodule update --remote
```
